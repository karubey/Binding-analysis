# Binding Analysis of Molecular Dynamics Simulations

This repository contains analysis workflows for molecular dynamics simulations, including Hamiltonian Replica Exchange (H-REMD), MBAR free energy calculations, and structural trajectory analysis.

## Features
- MBAR-based free energy estimation using pymbar
- Replica exchange trajectory sorting
- RMSD and structural analysis of simulation data
- Flap distance vs RMSD correlation analysis
- Publication-quality visualization scripts

## Requirements
- numpy
- matplotlib
- pymbar

## Usage
Run the main analysis script:
```bash
python md_mbar_analysis.py
